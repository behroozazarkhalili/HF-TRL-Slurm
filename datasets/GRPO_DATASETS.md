## Detailed Dataset Information

### 1. nvidia/OpenMathInstruct-2

| Property | Value |
|----------|-------|
| **Total Samples** | 14M problem-solution pairs |
| **File Size** | 12.6 GB |
| **Token Stats** | 564 questions (0.1%) exceed 1024 tokens |
| **License** | CC-BY-NC-4.0 |
| **Generator** | Llama-405B |
| **Source** | NuminaMath-derived problems |

**Description**: Large-scale math instruction dataset with 14M problem-solution pairs generated by Llama-405B. Most problems fit within 1024 tokens.

**Recommended Config**:
- Max Seq Length: 2,048
- Batch Size: 1-2
- Use streaming mode for efficiency

## Training Recommendations by Use Case

| Use Case | Recommended Dataset | Max Seq Len | Notes |
|----------|---------------------|-------------|-------|
| Quick baseline | orca-math-200k | 1,024 | Compact, fast training |
| Standard GRPO | NuminaMath-CoT | 2,048 | Well-tested, balanced |
| Large-scale | OpenMathInstruct-2 | 2,048 | Use streaming |
| Long reasoning | OpenR1-Math-220k | 16,384 | Requires large context |
| Tool-integrated | NuminaMath-TIR | 4,096 | Code execution traces |
| Competition-level | olympiads-ref | 8,192 | High-quality proofs |
| Comprehensive | NuminaMath-1.5 | 2,048 | Largest variety |
